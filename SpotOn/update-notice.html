SpotOn Update Notice

Welcome to the latest version (4.0.0) of SpotOn! We've made some significant changes to improve your experience.

Important: If you're upgrading from version 3.1.0 or lower, please note that many (and I mean many) features have been updated or changed. Some settings or options you previously had may have been removed or reset, it's recommended you start from scratch, sorry for any setbacks this has caused.

What's Changed?

  • Major UI improvements and optimizations
  • Massive performance enhancements
  • Updated settings structure
  • Updated and removed redundant css

What Should You Do?

We recommend reviewing and reconfiguring your settings to ensure everything is set up according to your preferences. Some of your previous settings may need to be adjusted to work with the new version.

PS: If something breaks, turn it off and on again, like a toggle, and or reload the page, 9/10 times it fixes the issue(s)!

What to Expect Next / Please Read.

Sadly, this cannot continue. I've spent the past five months (2 of which was me fixing issues, than resolving myself to a full rewrite is now required, considering how extensive my knowledge is in JS compared to the start.) rewriting the entire extension from the ground up-again. This means every single CSS file has been updated and changed, with new comments in each file explaining what it does. All of the JavaScript has been rewritten from scratch. As a result, SpotOn now loads in under 0.1ms, and the locking/unlocking process feels smoother. The entire settings menu has also been redesigned and overhauled to be more user-friendly.

I mentioned in one of the previous releases that I may never update this again, and it's very likely this will be the last major version before I stop working on it or add anything new. It's come to my attention that I've been developing this extension for over three years now. During that time, others have copied parts of my code and climbed higher in the extension store rankings. I never did this for fame, but it saddens me to see my work being profited from. To this day (23/05/2025), I haven't earned a single cent. I never created this for money-I'm just stating the fact that my work has been unpaid, and with all the time I've put in... man, I could've done so much more.

So now, SpotOn will only receive slower and minor CSS updates when something major breaks. Eventually, it might be removed from the store or fully archived as I shift my focus to other development projects, thank you for coming along in the journey with me, it's been a pleasure.

If you wish to support me, there is a link in the settings page, or on my GitHub as to how to sponsor, or one time send anything, although, I'll be frank, I do not believe anyone will, as this is, and always will be a free extension.
